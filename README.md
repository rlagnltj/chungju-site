# chungju web site
노약자들을 위한 충주 홍보 사이트

# 프로젝트 소개
- 충주시를 참고한 노약자들을 위한 충주 홍보 사이트입니다.

# 개발 기간
- 24.07.29일 - 22.08.01일

# 맴버구성
- 서명준 - html용어를 사용하여 웹사이트코드 작성
- 김휘서 - 충주관광지, 충주맛집, 충주산찾기 등 충주에 관한 자료조사

# 개발환경
- 'html'
